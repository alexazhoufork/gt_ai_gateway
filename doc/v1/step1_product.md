# Serverless AI Gateway - v1.0/v1.1 前端产品需求文档

## 1. 项目概述

### 1.1 产品背景

Serverless AI Gateway 是一个无服务器 AI 网关，用于统一管理和调度多个 AI 供应商（OpenAI、Anthropic、阿里云等）的 API 服务。前端管理界面是系统的管理控制台，供管理员进行配置管理和监控。

### 1.2 v1 版本目标

完成 AI 网关的基础管理功能，包括登录认证、用户管理、供应商管理和模型管理。

### 1.3 目标用户

| 用户类型 | 角色描述 | 主要需求 |
|----------|----------|----------|
| 系统管理员 | 负责网关配置和管理 | 用户管理、供应商配置、模型管理 |

## 3. v1.0 功能需求 - MVP 第一部分

### 3.1 登录认证（P0）

#### 3.1.1 登录页面
- **功能描述**: 管理员通过 Token 登录系统
- **输入项**:
  - Token: 文本输入框，必填，支持粘贴
- **交互逻辑**:
  - 输入 Token 后点击登录按钮
  - 验证 Token 有效性（调用后端 welcome 接口）
  - 验证成功：跳转至仪表盘
  - 验证失败：显示错误提示
- **Token 管理**:
  - Token 存储在 localStorage
  - 页面刷新后自动恢复登录状态
  - Token 过期/无效时自动跳转登录页

#### 3.1.2 登出功能
- **位置**: 顶部导航栏用户菜单
- **功能**: 清除 Token 并跳转登录页

---

### 3.2 基础框架（P0）

#### 3.2.1 技术实现
- Vue 3 + TypeScript + Vite 项目搭建
- Ant Design Vue 集成
- Vue Router 路由配置
- Pinia 状态管理配置
- Axios 封装（请求/响应拦截器）
- 全局错误处理

#### 3.2.2 页面布局
- **侧边栏 + 主内容区域布局**
- **顶部导航栏**: 包含 Logo、用户信息、登出按钮
- **侧边栏**: 菜单导航，包含各个功能模块入口

---

### 3.3 用户管理（P0）

#### 3.3.1 用户列表
- **展示字段**:
  - ID
  - 用户名
  - Token（支持显示/隐藏）
  - 用户类型（normal/admin）
  - 创建时间
  - 更新时间
- **搜索筛选**:
  - 按用户名搜索
  - 按用户类型筛选（normal/admin）
- **分页**: 支持页码切换和每页条数设置

#### 3.3.2 新建用户
- **触发方式**: 列表页"新建用户"按钮
- **表单字段**:
  - 用户名：必填，字符串
  - Token：选填，为空时后端自动生成
  - 用户类型：单选（normal/admin），默认 normal
- **验证规则**:
  - 用户名必填且唯一

#### 3.3.3 用户详情
- **入口**: 列表页点击用户名或"查看详情"
- **展示内容**:
  - 用户基本信息（ID、名称、类型）
  - Token（支持显示/隐藏、复制）
  - 时间信息（创建时间、更新时间）

---

### 3.4 系统状态（P1）

#### 3.4.1 仪表盘页面
- **功能描述**: 展示系统运行状态和核心指标
- **展示内容**:
  - 系统运行状态（正常/异常）
  - 简单的状态卡片

---

## 4. v1.1 功能需求 - MVP 第二部分

### 4.1 供应商管理（P0）

#### 4.1.1 供应商列表
- **展示字段**:
  - ID
  - 类型（openai/anthropic/aliyun/deepseek/google/other）
  - 名称
  - Token（支持显示/隐藏）
  - API 格式（OpenAI/Anthropic）
  - 创建时间
- **搜索筛选**:
  - 按名称搜索
  - 按类型筛选
  - 按 API 格式筛选
- **分页**: 支持页码切换

#### 4.1.2 新建供应商
- **表单字段**:
  - 类型：下拉选择（openai/anthropic/aliyun/deepseek/google/other）
  - 名称：必填，字符串
  - Token：必填，供应商 API Token
  - URLs：JSON 对象，配置 API 端点地址
- **验证规则**:
  - 名称、Token 必填
  - URLs 为有效 JSON 格式

#### 4.1.3 编辑供应商
- **功能**: 修改供应商配置
- **可编辑字段**: 名称、Token、URLs
- **限制**: 类型不可修改

#### 4.1.4 供应商详情
- **展示内容**:
  - 基本信息（ID、类型、名称）
  - Token（支持显示/隐藏、复制）
  - URLs 配置（JSON 展示）
  - 关联模型列表
  - 时间信息

---

### 4.2 模型管理（P0）

#### 4.2.1 模型列表
- **展示字段**:
  - ID
  - 模型名称
  - 供应商名称
  - 启用状态（启用/禁用）
  - 创建时间
- **搜索筛选**:
  - 按模型名搜索
  - 按供应商筛选
- **分页**: 支持页码切换

#### 4.2.2 新建模型
- **表单字段**:
  - 模型名称：必填，字符串
  - 供应商：下拉选择，关联供应商列表
  - 启用状态：开关，默认启用
- **验证规则**:
  - 模型名、供应商必填

#### 4.2.3 编辑模型
- **可编辑字段**: 模型名称、供应商、启用状态

#### 4.2.4 模型详情
- **展示内容**:
  - 基本信息（ID、名称）
  - 供应商信息
  - 启用状态
  - 时间信息

---

### 4.3 用户体验优化（P1）

#### 4.3.1 加载状态优化
- 表格数据加载时显示骨架屏
- 表单提交时按钮显示 loading 状态

#### 4.3.2 操作提示优化
- 操作成功显示绿色 success 提示
- 操作失败显示红色 error 提示，并展示错误信息
- 提示自动消失（3秒）

#### 4.3.3 空状态优化
- 表格无数据时显示友好的空状态提示
- 提供新建入口的快捷按钮

---

## 5. 非功能需求

### 5.1 性能要求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 首屏加载时间 | < 3s | 首次访问加载时间 |
| 页面切换时间 | < 500ms | 路由切换响应时间 |
| 列表加载时间 | < 1s | 100 条数据以内 |
| API 响应超时 | 30s | 超时错误处理 |

### 5.2 兼容性要求

| 浏览器 | 最低版本 |
|--------|----------|
| Chrome | 90+ |
| Firefox | 88+ |
| Safari | 14+ |
| Edge | 90+ |

### 5.3 安全要求

- Token 存储使用 localStorage
- API 请求使用 HTTPS
- 敏感信息（Token）默认隐藏，可切换显示
- 退出登录后清除所有本地存储

## 6. 用户界面流程

### 6.1 登录流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ 访问首页 │ -> │ 检查Token│ -> │ 无效/无  │ -> │ 登录页面 │
└─────────┘    └─────────┘    └─────────┘    └────┬────┘
                             │ 有效               │
                             v                    v
                          ┌─────────┐    ┌─────────┐
                          │ 仪表盘  │ <- │ 输入Token│
                          └─────────┘    └─────────┘
```

### 6.2 主要业务流程

```
登录成功
    │
    ├─> 仪表盘 ──> 查看系统状态
    │
    ├─> 用户管理 ──> 新建/查看用户
    │
    ├─> 供应商管理 ──> 新建/编辑/查看供应商
    │
    └─> 模型管理 ──> 新建/编辑/查看模型
```
