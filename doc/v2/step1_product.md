# Serverless AI Gateway - v2.x 前端产品需求文档

## 1. 版本概述

### 1.1 v2 版本目标

实现监控分析功能，包括请求记录查看、仪表盘统计和 API 测试工具，提升数据可视化和开发调试体验。

### 1.2 功能范围

| 版本 | 模块 | 目标 |
|------|------|------|
| v2.0 | 请求记录 | 实现请求记录查看和详情分析功能 |
| v2.1 | 仪表盘 + API 测试 | 增加统计数据和调试工具 |

---

## 2. v2.0 功能需求 - 监控分析第一部分

### 2.1 请求记录（P0）

#### 2.1.1 记录列表
- **展示字段**:
  - ID
  - 用户名称
  - 供应商名称
  - 模型名称
  - 状态（success/failed）
  - 创建时间
- **搜索筛选**:
  - 按状态筛选（success/failed）
  - 按时间范围筛选
  - 按用户名搜索
  - 按模型名搜索
- **分页**: 支持页码切换
- **自动刷新**: 支持开启/关闭自动刷新

#### 2.1.2 记录详情
- **基本信息**:
  - 请求 ID、用户、供应商、模型
  - 请求状态、时间信息
- **请求数据**:
  - JSON 格式展示
  - 语法高亮
  - 复制功能
- **响应数据**:
  - JSON 格式展示
  - 语法高亮
  - 复制功能
- **错误信息**: 失败时展示错误详情

---

## 3. v2.1 功能需求 - 监控分析第二部分

### 3.1 仪表盘增强（P0）

#### 3.1.1 统计数据展示
- **今日请求总数统计**
- **今日请求成功率统计**
- **活跃用户数统计**
- **活跃模型数统计**
- **最近请求快捷入口**

#### 3.1.2 自动刷新
- **刷新间隔**: 30秒
- **刷新控制**: 暂停/开始刷新按钮
- **数据时效性**: 显示最后更新时间

### 3.2 API 测试工具（P0）

#### 3.2.1 请求配置面板
- **API 格式选择**: OpenAI / Anthropic 格式切换
- **模型选择**: 下拉选择已启用的模型
- **消息输入**:
  - 文本区域输入消息内容
  - 支持多轮对话配置
- **参数配置**:
  - temperature: 0-2，默认 0.7
  - max_tokens: 1-8192
  - stream: 是否启用流式响应

#### 3.2.2 响应展示
- **普通响应**: 完整展示返回内容
- **流式响应**: 实时显示生成的内容
- **状态指示**: 显示请求状态（等待中/生成中/完成/错误）
- **操作按钮**: 停止生成、清除响应

#### 3.2.3 历史记录
- **存储方式**: localStorage
- **记录内容**: 请求配置、响应内容、状态、时间
- **管理功能**: 查看、加载、删除历史记录

### 3.3 用户体验优化（P1）

- 错误提示优化
- 加载动画优化

## 4. 非功能需求

### 4.1 性能要求

| 指标 | 目标值 |
|------|--------|
| 记录列表加载 | < 1s（100条数据） |
| 统计数据刷新 | < 500ms |
| API 测试响应 | 实时流式展示 |
| JSON 渲染 | 支持大 JSON 虚拟滚动 |

### 4.2 新增依赖

| 库 | 用途 |
|------|------|
| @antv/g2plot | 图表可视化（仪表盘统计） |
| vue3-json-viewer | JSON 语法高亮展示 |
| @microsoft/fetch-event-source | SSE 流式响应处理 |
